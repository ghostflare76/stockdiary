# StockDiary PRD (Product Requirements Document)
## 1. 프로젝트 개요
### 목적
- 투자자들의 주식 매매 기록을 효율적으로 관리하고 AI 기반 분석을 통해 투자 인사이트 제공
- 직관적이고 단순한 UI로 사용자 경험 최적화
### 타겟 사용자
- 개인 투자자
- 매매 기록을 체계적으로 관리하고 싶은 사용자
- 자신의 투자 패턴을 분석하고 싶은 사용자
### 2. 유저 플로우
1. 메인 화면
- Ticker 별 매매 기록 리스트 확인
- 신규 매매 기록 추가 버튼
2. 매매 기록 작성
- Ticker 선택/입력
- 매수/매도 선택
- 거래 가격, 수량 입력
- 매매 사유 및 메모 작성
3. 상세 보기
- 개별 매매 기록 상세 정보 확인
- AI 분석 결과 확인
- 수정/삭제 기능
### 3. 핵심 기능
매매 기록 관리
- Ticker 별 매매 기록 작성/수정/삭제
- 매수/매도 구분에 따른 시각적 표현
- 매수: 초록색 + ↑ 아이콘
- 매도: 빨간색 + ↓ 아이콘

리스트 뷰
- Ticker 별 그룹핑
- 날짜순 정렬
- 매매 유형별 필터링

AI 분석
- GPT API를 활용한 매매 기록 분석
- 투자 패턴 분석
- 개선점 제안
- 리스크 요인 분석

데이터 저장
- Local Storage를 활용한 데이터 저장
- JSON 형식의 데이터 구조
### 4. 기술 스택
프론트엔드
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- ShadcN UI

상태 관리
- Zustand

API 통신
-OpenAI API (GPT)
-axios

유틸리티
-date-fns (날짜 처리)
-zod (데이터 검증)

### 5. 향후 개선 사항
- 사용자 관리
- 회원가입/로그인 기능
- 사용자별 데이터 관리
- 데이터 관리
- 클라우드 데이터베이스 연동
- 데이터 백업/복구 기능
- 기능 확장
- 포트폴리오 분석
- 차트 시각화
- 실시간 주가 정보 연동
- 매매 기록 엑셀 추출
- 모바일 최적화
- AI 기능 강화
- 더 상세한 투자 분석
- 맞춤형 투자 조언
- 시장 동향 분석
